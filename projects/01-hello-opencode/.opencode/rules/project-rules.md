# Project Rules

## File Structure
```
/hello-opencode
├── index.html
└── style.css
```

## HTML Rules
- Always use `<!DOCTYPE html>` and `<html lang="en">`
- Include viewport meta tag
- Use semantic tags (header, main, section, footer)

## CSS Rules
- Mobile-first responsive design
- Use CSS variables for brand colors
- Prefer flexbox and grid
- Smooth transitions (0.3s ease)

## Brand
- Primary: #0074FF
- Background: #0a0e1a
- Text: #eef2ff
- Accent: #00ffcc

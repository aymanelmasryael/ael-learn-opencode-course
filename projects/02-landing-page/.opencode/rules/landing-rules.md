# Landing Page Design Rules

## Layout
- Header: sticky glassmorphism navbar with logo + nav links
- Hero: 100vh, centered content, background gradient, animated orb
- Features: 4-column grid, each card: icon + h3 + p
- Testimonials: horizontal scroll with snap points
- CTA: centered, wide button with box-shadow glow
- Footer: 3-column: brand info, links, social

## Responsive
- Desktop: 1024px+ (4-column features)
- Tablet: 768-1023px (2-column features)
- Mobile: <768px (1-column, stacked)

## Colors
- Nav: glass effect (bg: rgba(5,12,24,.7))
- Hero gradient: #000 → #0a1628
- Cards: bg rgba(5,12,24,.5), border rgba(0,116,255,.2)
